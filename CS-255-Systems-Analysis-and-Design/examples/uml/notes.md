# UML / ERD Notes

## 📌 Key Diagrams
- **Use Case Diagram**: Actors (Student, Admin, DMV) + system functions  
- **Class Diagram**: Entities like User, Exam, Schedule  
- **Sequence Diagram**: Steps for "Student registers & takes exam"  
- **ERD**: Tables → Users, Courses, Exams, Results
